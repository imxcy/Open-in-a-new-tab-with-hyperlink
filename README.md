# 超链新标签页打开 — 让所有链接在新标签页打开的 Chrome 插件

点击网页链接时，当前页面总是被覆盖跳走？这款插件让所有链接自动在新标签页打开，原页面始终保留。

## 功能

- 全局生效，适用于所有网站
- 点击链接文本、图片等子元素都能正确识别
- 自动跳过锚点、`mailto:`、下载等非导航链接
- **点击图标可快速开关**，关闭后恢复浏览器默认行为
- 安装即用，零配置

## 安装

1. 下载并解压插件压缩包
2. 打开 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择解压后的文件夹

## 使用

安装后自动生效。点击浏览器右上角插件图标，可通过开关快速启用/禁用，图标也会随之变色提示当前状态。
